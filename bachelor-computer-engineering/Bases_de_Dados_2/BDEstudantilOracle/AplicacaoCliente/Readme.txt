A aplicação cliente foi implementada em java e necessita da versão 1.4 ou superior do java.
A ligação à base de dados está configurada por defeito para o servidor "serv-lbd", no entanto é necessário aceder 
á opção "Configurar ligação" para introduzir o user e a password.
A aplicação permite a inserção de alunos e a visualização dos dados inseridos na tablela "tbl_aluno".
